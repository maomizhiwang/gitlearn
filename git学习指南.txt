新设备初始化：git --global user.name 你的githubid
	          git --global user.email 你的邮箱
git init 初始化仓库
添加文件 git add 文件名
上传文件 git commit -m 注释
上传文件到远程库 git push origin master
克隆远程库 git clone git@github.com:你的id/你的库名字.git //克隆后的文件在你 git bash所在目录下

添加分支 git branch 分支名字
切换分支 git checkout 分支名字
将上述操作合并操作 git checkou -b 分支名
